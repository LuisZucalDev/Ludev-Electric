## 🚀 Cómo Usar Ahora

### Ver el Sitio
```bash
# Simplemente abrir en navegador:
open index.html
# O en el navegador:
file:///tu/ruta/Ludev-Electric/index.html
```

### Editar Contenido
1. Abrir `index.html` en VS Code
2. Buscar la sección por ID (`#inicio`, `#servicios`, etc)
3. Editar el contenido
4. Guardar (Ctrl+S)
5. Recargar navegador (F5)

### Cambiar Colores
En `index.html`, dentro del `<script>` Tailwind Config:
```javascript
colors: {
  primary: {
    600: '#TU_NUEVO_COLOR',
  }
}
```
